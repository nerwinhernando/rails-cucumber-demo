# README

This repository is for demonstration of cucumber. Also it has the following gems.

* rspec
* database cleaner
* should-matchers
* factory_bot_rails

## Rails Setup

* Ruby Version 2.6.3
1. rvm install 2.6.3
* Rails Version 6.0.1
1. bundle install

## Instructions
1. bundle exec rails db:create
2. bundle exec rails db:migrate
3. rails s
